## Features
- [x] Login
- [x] Send & receive messages
- [x] Typing indicator
- [x] Send Files
- [x] Persist User, Messages and rooms using local database
- [x] ChatMarker (In Progress)
- [ ] Group Chat
- [ ] Presence
- [ ] File Previews
- [ ] Push Notifications When App is in Background.
